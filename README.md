## Run it Locally

Please follow the below instructions to run this project in your machine:

1. Clone this repository

   ```sh
   git clone https://github.com/sagormajomder/chapterly-backend.git
   ```

2. Open the directory "backend" into visual studio code
3. Open Terminal and run `npm i` to install all dependencies
4. Create .env file into the root directory and write `PORT`, `DB_USER`, `DB_PASS` and `FIREBASE_SERVICE_KEY` environement variable to run the project.
5. Run `npm run devServer` to run the project server

The project will be available on http://localhost:5000/ by default.
